# YOLOv11 Project

This project uses YOLOv11 for object detection tasks.

## Files

- `main.py` - Main application script
- `yolo11n.pt` - YOLOv11 nano model weights

## Usage

Run the main script:

```bash
python main.py
```

## Requirements

- Python 3.7+
- YOLOv11 library and dependencies

## Installation

Install required dependencies:

```bash
pip install -r requirements.txt
```
